## Sobre o Projeto

O **Programa Hotel Bela Vista** é um ecossistema requisitado pelo curso do Senai. Ele foi criado para melhorar a qualidade de vida dos trabalhadores, que frequentemente faziam relatórios em papel, o que atrasava o trabalho. A ideia é que, com o auxílio de computadores e de um servidor próprio, o serviço se torne menos manual e mais simples por ser digital, além de apresentar baixo risco de perda de dados.
